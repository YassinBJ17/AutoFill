
# OpenSc installation

## Download the latest release from https://github.com/OpenSC/OpenSC/releases

## Install required package
  sudo apt-get install pcscd libccid libpcsclite-dev libssl-dev libreadline-dev autoconf automake build-essential docbook-xsl xsltproc libtool pkg-config

## Unzip OpenSc, build and install it
  tar xfvz opensc-*.tar.gz
  cd opensc-*
  ./bootstrap
  ./configure --prefix=/usr --sysconfdir=/etc/opensc
  make
  sudo make install



# libp11 installation

  sudo apt install pkgconf libssl-dev
  ./configure
  make
  sudo make install


# p11tool installation
  sudo apt install gnutls-bin
